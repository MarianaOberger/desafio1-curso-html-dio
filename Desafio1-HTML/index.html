Estrutura básica do arquivo HTML.

(Voltar)

Convencionalmente nomeamos a página principal de "index", abrimos com uma tag chamada "html". As tag se inicia com "<" e termina com ">", isto para a tag ue inicia um bloco, a tag que finaliza deve ter uma barra logo após o sinal de "<". Para visualizarmos como nossa página esta se desenvolvendo podemos clicar com o botão direito dentro do VScode e seleciona a opção "Open with live server".

Atributos HTML.

(Voltar)

Atributos HTML são palavras especiais que possibilitam a alteração de configurações em determinadas partes do texto, é possivel alterar a largura de um campo, bem como a tipografia e outras caracteristicas do código. Os atributos são especificados dentro da tag que inicia a chamada, como por exemplo para linkarmos, como feito nesta página, utilizamos a identificação da tag para qual o link deve direcionar o usuário e colocamos essa identificação na tag que deve ser selecionada.

Tags e seus atributos.

(Voltar)

Existem três partes principais das tags em HTML: tag de abertura, que geralmente indica o tipo de conteúdo ou a caracterização do que se segue, o conteúdo que será oque o usuário pode vizualizar e selecionar e a tag de fechamento que encerra as atribuições da tag de abertura (nem todas as tags precisam ser fechadas obrigatoriamente, como no cado da tag "hr" que utilizamos no inicio da codificação desta página). O uso de tags de forma correta permite ao navegador interpretar a diferença entre as utilidades que o programador deseja que o usuário possa realizar.

Tipos de tags existentes:

(Voltar)
  • Tags estruturais: mudam o posicionamento dos elementos na página;
  • Tags de cabeçalho: alteram as parte de cabeçalho da página, como seu título, descrição e favicon;
  • Tags de links: gerenciam os links da página;
  • Tags de listas: permitem a criação de listas ordenadas ou não;
  • Tags de formatação de caracteres: adicionam formatação especial aos textos;
  • Tags de manipulação de elementos:melhoram o controle sobre o posicionamentos dos itens na página;
  • Tags de conteúdo multimídia: servem para a inserção de conteúdos como vídeos e imagens;
  • Tags de frames: para fazer a divisão da página internamente;
  • Tags de formulários: cria formulários para os usuários;/li>
  • Tags de tabelas: cria e modifica tabelas.
